Other Project

Automatic Vacuum Cleaner

Pao-er Cleaner is the automatic robot cleaner that picks up the dust particles from the floor. The system is built using Arduino duemilanove, dc motors, ultrasonic sensors, motor controller and power supply. The sensor enables the automatic movement of the system. This system moves around the floor on its own avoiding the obstacles and cleaning the floor simultaneously.
